ABOUT THE PROGRAM

The robotics teams of Katy High School and Taylor High School have joined forces in order to provide an oppurtunity for aspiring computer scientists and engineers to learn all of the skills needed to function in those careers in a fun yet effective fashion: VEX Robotics! Our camp consists of the construction of a VEX robot from beginning to end, as well as a competition that pits robots with and against each other. Our program is designed to encourage and nuture the following fundamental skills:

settings

Engineering Design

A robotics team is nothing without a robot. Students will learn to solve problems by desigining and engineering a VEX robot.

code

Computer Science

A robot cannot move without code. Computers will be provided so that students can learn how to code their robot in either block code or Python.

group

Teamwork

Robots are not built alone. Participants will learn several teamwork skills, including communication and time management.
